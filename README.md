# 水果农产品商城系统简介 python669

## 项目概述
本系统是基于Python语言开发的水果农产品商城系统，采用了Django框架和Vue.js进行前后端开发，使用MySQL作为数据库管理，实现了在线购买、销售和管理水果农产品的功能。

## 技术栈
- 后端：Python + Django
- 前端：Vue.js
- 数据库：MySQL

## 系统角色
- 管理员：负责系统管理和维护，包括商品管理、订单处理等。
- 用户：在商城系统中进行浏览、购买和评价等操作。

## 系统特性
### 前后端分离
系统采用了前后端分离的架构，前端使用Vue.js，后端使用Django，通过API接口进行数据交互。
